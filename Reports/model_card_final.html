Model Card for Hotel Booking Dataset

Model Details

Overview

This model predicts whether a person would cancel a hotel booking or not

Version

name: Version 1.0
date: 2021-11-25

Owners

References

Considerations

Intended Users

  • Hotel and motel owners
  • ML researchers

Use Cases

  • Intended to be used for predicting probability of cancellation, understand contributing factors to bookings and develop mitigation strategies
  • Not intended to make judgements about sensitive segments of the society based on the cancellation predictions

Limitations

  • This dataset covers only the North American region & might not be representative of the common use cases and concerns covering other regions & hotel booking systems.

Ethical Considerations

  • Risk: Personal data is being used for the analysis, so privacy concerns could be raised
    Mitigation Strategy: Masking personal information and/or using encrypted values can help mitigate this issue
